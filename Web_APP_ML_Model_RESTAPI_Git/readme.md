> Install latest python.

> Install python packages:
	> pandas
	> joblib
	> flask
	> flask_restful

> Install Nodejs, Hyper terminal and set them up for running web applications.

> To install packages used in app.js open hyper terminal in same directory as app.js and use 'npm install' command.

> Packages used in app.js:
	> body-parser
	> ejs
	> express
	> request

> After installing all required Packages for app.js and api.py open two hyper terminal tabs
  and run commands node 'app.js' in one tab and 'python api.py' in another.
